Register Cheque Information For POS Orders.

This apps allows to register cheque information in pos orders based on configure payment methods.

Don't worry we will provide free 90 days support for any doubt, queries and bug fixing (excluding data recovery) Or Any type of issues related to this module. This is applicable from the date of purchase.

For safety purpose kindly test this module in your staging or development instance before using to production environment.

How to configure...?

-> Open payment methods from point of sale menu.

-> You can see checkbox for cheque registration to active for this payment method.

-> Now, Go to POS Dashboard

-> Select session which you want to display cheque information.

-> Now, Go to settings for that session.

-> You can see cheque information setting, Click to active.

-> Now open pos session, Add products in the cart.

-> Now you can see list of payment methods, Here you can select cheque registered payment method.

-> Now you can see bank icon, Click that icon to open new screen for cheque registration.

-> You can feel information about cheque.

-> In receipt, You can see cheque information.

-> In backend side open that pos order, You can see cheque information.
